# 🏫 โครงการพัฒนาเว็บไซต์: โรงเรียนบ้านท่าซุง (Baan Tha Sung School)

**สถานะโปรเจกต์:** เริ่มต้นพัฒนาระบบ Front-End  
**เป้าหมาย:** เป็นศูนย์กลางข้อมูลที่เข้าถึงง่ายสำหรับชุมชน ผู้ปกครอง และบุคลากร

---

## 🎨 1. Design Tokens (ระบบสีและตัวอักษร)

เพื่อให้เว็บไซต์คุมโทน (CI) ตามตราสัญลักษณ์ของโรงเรียน และสร้างประสบการณ์ใช้งานที่เป็นมิตร

### Colors (จานสี)

- **Primary Color:** `Deep Forest Green` (อ้างอิงสีเขียวจากริบบิ้นตราโรงเรียน) - ใช้สำหรับ Header, Footer และเน้นหัวข้อหลัก
- **Accent Color:** `Golden Yellow` (อ้างอิงสีเหลืองจากวงแหวนในโลโก้) - ใช้สำหรับปุ่ม Call to Action (CTA) และไฮไลต์สำคัญ
- **Background Color:** `Off-White` / `Cream` - เป็นสีพื้นหลังหลักเพื่อความสบายตา
- **Text Color:** `Dark Gray` - ใช้สำหรับเนื้อหาทั่วไป ลดความล้าของสายตา

### Typography (แบบอักษร)

- **Font Family:** แนะนำ `Prompt` หรือ `Sarabun`
- **Scale:** เน้นขนาดตัวอักษรที่ใหญ่กว่ามาตรฐานเล็กน้อย (Large Text) เพื่อรองรับผู้ปกครองและผู้สูงอายุในชุมชนให้สามารถอ่านผ่านสมาร์ทโฟนได้ง่าย

---

## 🏗️ 2. Tech Stack & Component Architecture

การออกแบบโครงสร้าง UI สำหรับการพัฒนาด้วย Vue.js (หรือร่วมกับ UI Framework เช่น Quasar)

### โครงสร้างหน้าหลัก (Homepage) ประกอบด้วย:

- **`AppHeader.vue`**
  - แสดงโลโก้โรงเรียนและชื่อโรงเรียนทางซ้าย
  - เมนูนำทาง (Navbar)
  - มี Dropdown Menu สำหรับ **"ข้อมูลเพิ่มเติม"** ซึ่งภายในประกอบด้วย:
    - ประวัติโรงเรียน และตราสัญลักษณ์
    - คลิปวีดีโอแนะนำโรงเรียน
    - คณะกรรมการบริหาร และที่ปรึกษาผู้อำนวยการ
    - ทำเนียบครู
    - ผังโครงสร้างการบริหารงานโรงเรียน
    - แผนการเรียน ระดับประถม

- **`HeroSection.vue`**
  - ภาพแบนเนอร์กว้างเต็มจอ (Full-width) เป็นภาพนักเรียนทำกิจกรรม
  - ข้อความพาดหัว: ชื่อโรงเรียน และสโลแกน _"เรียนดี วินัยเด่น เน้นคุณธรรม"_
  - ปุ่ม CTA สีเหลืองเด่นชัด

- **`NewsBoard.vue`**
  - แสดงข่าวสารและประกาศของโรงเรียนในรูปแบบ Card Grid

- **`SchoolCalendar.vue`**
  - ปฏิทินกิจกรรมโรงเรียน จัดรูปแบบ List หรือ Timeline ที่ดูง่าย ไม่ซับซ้อน

- **`ActivityGallery.vue`**
  - อัลบั้มภาพกิจกรรม (Masonry หรือ Grid Layout)

- **`AppFooter.vue`** (บังคับเลย์เอาต์ 3 คอลัมน์)
  - **คอลัมน์ซ้าย:** โลโก้, ข้อมูลติดต่อโรงเรียน (สพป.ฉะเชิงเทรา เขต 2), เบอร์โทรศัพท์, อีเมล, และปุ่ม CTA "ติดต่อโรงเรียน"
  - **คอลัมน์กลาง:** ไอคอนนักเรียน, ข้อมูลติดต่อฝ่ายธุรการ, และปุ่ม CTA "ระบบรับสมัครนักเรียน"
  - **คอลัมน์ขวา:** ปุ่ม CTA "ส่งข้อเสนอแนะ" (จัดชิดขวาบน)
  - **ส่วนล่างสุด:** แถบลิงก์โซเชียลมีเดีย และข้อความลิขสิทธิ์ `© 2026 โรงเรียนบ้านท่าซุง. All rights reserved.`

## ⚠️ 3. Strict Rules (กฎการออกแบบที่ต้องยึดถือ)

1.  **Accessibility First:** ต้องมี Contrast ของสีที่ชัดเจน ห้ามใช้ตัวอักษรสีอ่อนบนพื้นสว่างเด็ดขาด
2.  **Action Buttons:** ปุ่มที่ต้องการให้ผู้ใช้งานกด (เช่น สมัครเรียน, ติดต่อเรา) ต้องใช้สี `Golden Yellow` เพื่อสร้างจุดสนใจหลัก (Focal point)
3.  **Simplicity:** หลีกเลี่ยงการใช้ Animation ที่ซับซ้อนหรือ Pop-up เน้นความเร็วในการโหลดและการจัดเรียงข้อมูลที่ตรงไปตรงมา
